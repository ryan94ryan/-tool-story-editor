📖 使用說明

🚀 快速開始

本地使用：匯入主檔 → 編輯 → 儲存主檔 → 一鍵釋出

雲端同步：設定 GitHub Token → 雲端存檔/讀取（跨裝置）

☁ 雲端設定（GitHub Gist）

  1. 前往 GitHub Token 設定頁
  2. 點擊 Generate new token (classic)
  3. 勾選 gist scope → 產生 token
  4. 回到編輯器「☁ 雲端 → 設定 GitHub Token」貼上

存檔無大小限制，且支援版本歷史。

🖱 操作方式

左鍵點擊節點選取節點
左鍵拖曳節點移動位置（多選時同步移動）
右鍵拖曳框選多個節點
右鍵點擊空白取消所有選取
Ctrl + 滾輪（編輯器內）調整字體大小
ESC關閉彈窗

📝 指令語法說明

所有指令用半形大括號 {} 包裹，不可使用全形。

背景 / CG

{SetBG_名稱}設置背景圖（純指令行，null 為清除）
{CBG_名稱}更換背景（可嵌入對話行）
{CCG_名稱}顯示 CG 全屏圖

角色立繪　位置：0=左 1=中 2=右

{Call_X:角色ID}呼叫角色到位置 X（不自動動畫）
{Close_X}關閉位置 X 的角色
{SlideIn_X}從預設方向滑入（需先 Call）
{SlideIn_X_L/R/U/D}從指定方向滑入
{SlideOut_X}向預設方向滑出
{SlideOut_X_L/R/U/D}向指定方向滑出
{FadeIn_X}淡入顯示
{FadeOut_X}淡出消失

燈光控制

{LC0_Light} {LC0_Dark}位置 0 亮起 / 變暗
{LC1_Light} {LC1_Dark}位置 1 亮起 / 變暗
{LC2_Light} {LC2_Dark}位置 2 亮起 / 變暗
{LCbg_Light} {LCbg_Dark}背景亮起 / 變暗

音效 / 語音

{Aubgm_X}播放 BGM（X 為音樂編號）
{Ausfx_X}播放音效（X 為音效編號）
{AuStop_0}停止 BGM
{Voice_X}播放語音（X 為語音編號）
{ClearVoice}清除當前語音

特效 / 文字特效

{Shake_screen}畫面震動（地震、爆炸）
{Shake_box}對話框震動（驚訝、衝擊）
{TextShake}文字開始震動（憤怒、恐懼）
{TextWave}文字開始波浪（魔法、夢境）
{TextStop}停止文字特效

介面 / 延遲

{Tbox_Open}顯示對話框
{Tbox_Close}隱藏對話框
{Delay_X}等待 X 秒後繼續

旗標操作

{setflag:名稱:值}將旗標設為指定值
{addflag:名稱:值}將旗標累加（值可為負數）
{GetBook_欄位名稱}玩家獲得秘籍

流程控制

{G_X}跳轉到 Group_X
{S_X}顯示選擇組 SE_X
{IF_條件:G_T:G_F}條件分支（成立→G_T，否則→G_F）
{GOSCENE_X}跳轉場景 X（完成對話後 2 秒自動執行）
{BattleSetting_XXX}設定戰鬥 ID 並進入戰鬥
{GOCHAPTER_X}章節轉場動畫
{LOADCHAPTER_X}載入第 X 章
{LOADEXTRA_X}載入支線 Group X
{RETURNMAIN}從支線返回主線（+1 回合）
{RETURNMAIN_X}返回主線並跳到 Group X（+1 回合）

特殊指令

{gameover}顯示 Game Over 面板，停止對話
{non_reduceevent}此段劇情結束後不增加回合數（搭配 RETURNMAIN 用）
{addcurrentround}手動 +1 回合（eventround）
{Achievement_成就ID}解鎖成就（首次觸發彈窗）
{FUNC:swipe:角色:背景}切換場景過場動畫

全螢幕黑幕

{ScreenFadeOut_秒數}全螢幕淡出至黑幕（秒數支援小數）
{ScreenFadeIn_秒數}全螢幕從黑幕淡入回復

ExDialogue 全螢幕文字場景

{exdialogue_1}開啟 ExDialogue（全螢幕文字累積模式）
{exdialogue_0}關閉 ExDialogue（淡出）
{set_exbg_名稱}設定 ExDialogue 背景
{ex_clear}清空 ExDialogue 累積文字
{blank}插入空白行（ExDialogue 文字分段）

選擇節點（SE）格式

Buttonset_X:選項文字:{G_Y}選項 X，點擊後跳轉到 Group_Y
Buttonset_X:選項{f_旗標=值}:{G_Y}帶條件的選項（不符則隱藏）
Buttonset_X:null空按鈕（不顯示）

💡 小技巧

  • 輸入 / 叫出指令面板：編輯器內按 / 會彈出指令選單，打字篩選 → ↑↓ 選擇 → Enter 插入
  • 在 Group 第一行寫 //簡短大綱 可作為節點標籤
  • 編輯器的「折疊」模式可隱藏指令，只顯示對話
  • 節點位置會隨雲端存檔一起保存
  • {non_reduceevent} 放在劇情任意位置即可生效，結束後不推進時段
  • {GOSCENE_X}、{RETURNMAIN} 前的對話顯示完後等 2 秒才自動跳轉
